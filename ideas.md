# 《艰苦卓绝》Minecraft 服务器官网 - 设计理念

## 项目背景
- **服务器名称**：艰苦卓绝
- **成立时间**：2022年1月18日
- **核心体验**：前期高难度 + 后期高自由度
- **模组特色**：以 Create（机械动力）为核心，涵盖魔法、冒险、农业等
- **加入方式**：QQ群 657699707

---

## 设计方向选择

### 选中方案：工业暗黑科技美学（Industrial Dark Tech Aesthetic）

**设计哲学**：将 Minecraft Create 模组的机械美学与现代科技感结合，打造一个充满工程感、精密感的官网。通过暗黑色调、冷色调、几何线条和工业元素，传达服务器的"艰苦卓绝"精神——高难度、高自由度、精密工程。

---

## 核心设计原则

1. **工程精密感**：采用严谨的几何布局、精确的间距、清晰的模块化结构，体现 Create 模组的机械精密性
2. **暗黑科技感**：深色背景（深灰、深蓝、纯黑）搭配冷色调高亮（青蓝、荧光绿、冷白），营造未来科技感
3. **层次递进**：从首页的宏观介绍到模组特色的微观细节，逐层深化用户理解
4. **动态工业感**：通过微妙的动画、渐变、光效，让静态页面呈现机械运转的生命力

---

## 色彩哲学

### 主色板
- **背景色**：`#0a0e27`（深蓝黑）- 夜间工业厂房的既视感
- **次背景**：`#141829`（深灰蓝）- 卡片、模块背景
- **强调色**：`#00d4ff`（青蓝）- 科技感高亮，代表能量流动
- **辅助高亮**：`#39ff14`（荧光绿）- 工业警示色，用于重点信息
- **文本色**：`#e8eef5`（冷白）- 高对比度，易读性强

### 情感意图
- 深色传达**专业、严肃、高难度**的游戏体验
- 冷色调强化**科技、精密、工程**的氛围
- 青蓝 + 荧光绿的组合体现 **Create 模组的机械美学**

---

## 布局范式

- **非对称网格布局**：避免居中单调，采用左右错落的卡片布局
- **分层模块设计**：首页 → 核心特色 → 模组展示 → 加入方式，逐层递进
- **斜线分割**：使用 SVG 斜线分割不同区块，增强动态感
- **悬浮卡片**：卡片具有浮动感和深度，通过阴影和背景色区分

---

## 签名元素

1. **机械齿轮图案**：作为装饰元素出现在页面角落、卡片背景
2. **流动线条**：代表能量流动的渐变线条，连接不同区块
3. **网格背景**：微妙的网格纹理作为背景，强化工业感

---

## 交互哲学

- **悬停效果**：卡片悬停时发光、浮起，传达能量激活感
- **点击反馈**：按钮点击时有脉冲光效，体现机械启动感
- **滚动动画**：元素随滚动逐渐显现，营造渐进式探索体验
- **加载动画**：使用齿轮旋转、能量流动等动画，强化工业主题

---

## 动画指南

- **过渡时间**：300-500ms，保持流畅感
- **缓动函数**：`cubic-bezier(0.4, 0, 0.2, 1)` - 科技感的加速度
- **关键动画**：
  - 卡片进入：从下方淡入 + 微妙上移
  - 悬停效果：发光边框 + 背景亮度提升
  - 滚动触发：元素从两侧滑入 + 淡入

---

## 排版系统

- **标题字体**：`Orbitron` 或 `Space Mono`（等宽、科技感强）
  - H1：48px, 700 weight, 字间距 +2px
  - H2：32px, 600 weight, 字间距 +1px
  - H3：24px, 600 weight

- **正文字体**：`Inter` 或 `Roboto`（清晰易读）
  - 正文：16px, 400 weight, 行高 1.6
  - 小文本：14px, 400 weight, 行高 1.5

- **强调文本**：使用 `#00d4ff` 或 `#39ff14` 高亮关键词

## 页面结构规划

1. **首页 (Hero Section)**
   - 大标题 + 服务器简介
   - Create 模组的炫彩背景图
   - CTA 按钮：加入服务器

2. **核心体验区**
   - 前期高难度 vs 后期高自由度对比卡片
   - 动画展示游戏进度曲线

3. **模组特色展示**
   - Create（机械动力）- 主卡片
   - 魔法、冒险、农业 - 副卡片
   - 每个卡片包含图片 + 描述 + 特色亮点

4. **加入方式**
   - QQ 群二维码 / 群号
   - 服务器信息卡片
   - 成立时间展示

5. **页脚**
   - 简洁的联系信息
   - 社交链接
